# ClotherShoppiingSystem
衣服售卖系统
使用java基础只是开发的衣服售卖系统，代码是看千锋教育视频手敲的（ps:之前问客服代码无果后）
具体的实现的功能有：
1.登陆注册功能
2.购物功能
3.查看订单明细
4.显示商品功能
5.根据订单id查找某个订单
